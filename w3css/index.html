w3.css 更小，更快，更易于使用

W3
CSS

w3.css 是一个现代的，响应迅速的，移动优先的 CSS 框架。

w3.css 为所有浏览器提供平等：Chrome，Firefox，Edge，IE，Safari，Opera。

w3.css 为所有设备提供平等：Desktop，Laptop，Tablet，Mobile。

w3.css 仅是标准的 CSS（没有 jQuery 或 JavaScript 库）。

如何使用

要在您的网站上使用 W3.CSS，只需在您的网页上添加指向“w3.css”的链接：

<link rel="stylesheet" href="https://www.w3schools.com/w3css/4/w3.css">

由于国外网站可能会不稳定，故而，可以使用我的副本：

<link rel="stylesheet" href="https://xinetzone.github.io/w3css/4/w3.css">
